Un client Android NHentai non ufficiale. Questa app funziona per dispositivi API 14 (Android 4.0) e versioni successive.

Funzionalità API: sfoglia la pagina principale 

- Ricerca per query o tag 
- Includi o escludi tag 
- Sfoca o nascondi i tag esclusi 
- Scarica gallerie 
- Gallerie preferite 
- Abilita il PIN per accedere alle funzionalità personalizzate dell'app 
- Condividi gallerie 
- Apri nel browser 
- Segnalibro 
- Possibilità di mascherare l'app da calcolatrice
